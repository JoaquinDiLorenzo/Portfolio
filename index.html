Sobre mi

Apasionado por la programación y el desarrollo de software. Desde 2024 trabajo en proyectos reales: desde sitios web en producción hasta aplicaciones de análisis de datos con Python, pandas y matplotlib. Curso 3er año de Licenciatura en Sistemas en la UNLP y participo en el equipo de programación del Club de Finanzas UNLP, donde combino análisis de datos y economía. Trabajo con Python (Django, análisis de datos), Java (POO, patrones de diseño) y bases de datos como PostgreSQL. Me motiva resolver problemas y transformar ideas en soluciones funcionales.

Proyectos recientes

Club de Finanzas UNLP

Sitio web institucional desarrollado para el Club de Finanzas UNLP. Permite la gestión de miembros, eventos y la publicación de artículos sobre finanzas, economía e inversiones. Incluye panel de administración seguro, autenticación y base de datos relacional.

  • Tecnologias Utilizadas - Django · PostgreSQL · HTML · CSS · Bootstrap
  • Ver Online - https://www.clubdefinanzasunlp.com/

Alquil.ar

Proyecto realizado para la materia "Ingeniera de Software 2" para la gestión integral de alquiler de maquinarias. Incluye sistema de reservas, control de stock, penalizaciones por entregas tardías o roturas, integración con Mercado Pago para cobros,etc.

  • Tecnologias Utilizadas - Django · PostgreSQL · Google Maps API · Mercado Pago API - Bootstrap
  • GitHub - https://github.com/FedericoKesselman/alquil.ar
  • Más Informacion y Demo - Ver publicacion de Linkedin

Letterboxd Recommender

Sistema de recomendación de películas basado en machine learning, que utiliza datos de la plataforma Letterboxd. Procesa valoraciones y preferencias del usuario para sugerir películas personalizadas. También cuenta con una sección llamada "Movie Night" la cual cruza los datos de dos personas y recomienda peliculas.

  • Tecnologias Utilizadas - Python · Pandas · Scikit-learn · Jupyter Notebook
  • GitHub - https://github.com/baltumenvielle/letterboxdrecommender
  • Más Informacion y Demo - Ver publicacion de Linkedin
  • Ver Online - https://letterboxdrecommender.streamlit.app/

Project Tree
